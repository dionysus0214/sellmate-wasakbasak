## 백엔드 환경구성 가이드
python v3.9 설치
pip 설치

```
> cd backend
> pip install -r requirements.txt
> uvicorn main:app --reload
```

## 프론트엔드 환경구성
1. react 설치
2. mui 설치(react UI framework)
```
> cd frontend
> yarn add @mui/material @emotion/react @emotion/styled
```
react >= 17.0.0 및 react-dom >= 17.0.0 에서 적용
